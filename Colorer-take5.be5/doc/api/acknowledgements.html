Acknowledgements

HRC database

I'm working on maintenance of the HRC database with help of many people. You can find their acknowledgements in corresponding HRC file headers. Most 'named' HRC's has their author's contact email or url.

Unzip code

Colorer library uses minizip source codes, written by Gilles Vollant and based on Jean-loup Gailly and Mark Adler zlib library.
You can find latest minizip source codes on http://www.winimage.com/zLibDll/unzip.html, or in zlib distribution package http://www.gzip.org/zlib/zlib114.zip.
Colorer uses minizip v0.20, found at http://www.winimage.com/zLibDll/unzip20.zip.

dtd2xsd.pl

Colorer package includes dtd2xsd script, used for transformation of DTD document descriptions into XML Schema form. This perl script is written by Mary Holstege, Yuichi Koike, Dan Connolly, Bert Bos.
I've modified it and fixed some minor errors and problems.
